# Rotas:

**/login**
post: envie um username para receber o token

**/order**
post: envie o payload para cadastrar o pedido

**/order/list**
get: retorna todos os pedidos cadastrados

**/order/:numeroPedido**
get: retorn o pedido especificado
put: atualiza o pedido especificado
delete: deleta o pedido especificado
